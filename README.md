# Variation Surveys

Survey of recent (2025-2026) high-profile pathogen genomic studies with publicly available sequencing data suitable for variant calling analysis using [BRC Analytics](https://brc-analytics.org).

## Goal

Identify studies where simple variant calling can yield insights and highlight BRC Analytics functionality. Studies must have clearly stated BioProject/SRA accessions.
