# Candidate Pathogen Survey Papers with BioProject IDs (2024-2026)

Curated list of high-profile publications with clearly stated BioProject accessions, suitable for demonstrating variant calling analytics.

## 1. Candida auris -- Global Genomic Epidemiology (12,644 genomes)

| Field | Detail |
|-------|--------|
| **Title** | Global genomic epidemiology of *Candida auris*: analysis of 12,644 whole genome sequences from 1997-2024 |
| **Authors** | Gifford H ... Farrer RA |
| **Journal** | bioRxiv (2026) |
| **DOI** | 10.64898/2026.02.03.703534 |
| **Pathogen** | *Candida auris* (WHO critical priority fungal pathogen) |
| **BioProject** | **PRJNA1372636** (new isolates); compiled from many public BioProjects |
| **Sample size** | 12,644 WGS |
| **Variant calling interest** | CNV in azole target *ERG11* (10-15 copies); 18 novel *FKS1* hot-spot echinocandin resistance variants; clade structure across 6 clades; resistance variants enriched in urine vs blood |
| **Why interesting** | Largest *C. auris* genomic dataset; fungal AMR is underrepresented in variant-calling demos; clear clinical relevance |
